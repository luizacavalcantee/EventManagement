Event Management System

Luiza Cavalcante, Clara Oliveira Costa, Amós Santana, Saunay Coutinho

Gerenciamento de Eventos Simplificado com C++

Uma solução de back-end completa para plataformas de eventos, permitindo criar, gerenciar e deletar eventos e participantes através de uma API RESTful robusta.

Visão Geral das Funcionalidades

Gerenciamento Completo de Eventos

Crie, edite, visualize e delete eventos com total controle e validação de dados. O painel administrativo oferece uma visão clara de todos os eventos cadastrados, facilitando a gestão.

Gestão de Participantes

Inscreva, edite, liste e remova participantes de eventos específicos. O sistema controla a capacidade máxima, garantindo que os eventos não fiquem superlotados e oferecendo uma experiência organizada.

Dashboard com Estatísticas

Acesse um dashboard com estatísticas em tempo real, incluindo o número total de eventos, a quantidade de participantes inscritos, e uma visualização rápida de eventos futuros e que ocorrem no dia.

Tecnologias e Arquitetura

O sistema foi construído com uma arquitetura de 3 camadas, utilizando C++ puro no backend para máxima performance. A comunicação é feita via API RESTful, com persistência de dados em um arquivo de texto local para simplicidade e portabilidade.

  • Backend: C++ puro com a biblioteca Winsock para networking.
  • Frontend: JavaScript (ES6+), HTML5 e CSS3 com Bootstrap 5.
  • Princípios: Forte uso de Programação Orientada a Objetos (Herança, Encapsulamento, Polimorfismo).
